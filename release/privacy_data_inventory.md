# Privacy And Data Inventory

## Local Data

The app stores the following data on device only:

- Current level.
- Highest unlocked level.
- generatorVersion.
- Tutorial completion status.
- Acknowledged rule codes for rule introductions.
- Sound setting.
- Haptic setting.

The app does not currently store:

- User account information.
- Email address.
- User-generated content.
- Play time.
- Personal best records.
- Cloud sync data.

## External SDKs

- Google Mobile Ads SDK is included for interstitial ads.
- UMP consent functionality is used through `google_mobile_ads`.

The exact data collection and sharing disclosures for these SDKs must be checked against the latest Google Mobile Ads and store policy documentation before submission. This document does not claim that the SDKs collect no data.

## Features Not Used Directly By App Code

- Sign-up or login.
- First-party server.
- Cloud storage.
- Location.
- Contacts.
- Camera.
- Microphone.
- Payments.
- User-generated content.

## Manual Store Disclosure Items

- Google Play Data Safety.
- App Store App Privacy.
- Advertising SDK disclosure.
- Tracking and consent answers.
- Privacy policy URL reachability and content review.

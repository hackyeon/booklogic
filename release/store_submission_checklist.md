- [ ] Families policy applicability.
- [ ] Test account requirement.
- [ ] App category.

## Screenshot Checklist

- One-tier basic puzzle.
- Bookshelf with clue cards.
- Two-tier puzzle.
- Three-tier puzzle.
- New vertical clue example.
- Clear result screen.
- Tutorial or primary interaction guide.

Screenshots must not include debug UI, internal IDs, or test ad creatives.
